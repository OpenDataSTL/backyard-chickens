About

The purpose of this project is to better communicate chicken regulations by municipality to the people of Saint Louis and Saint Louis County based on local municipality zoning regulations.
This project was built by the Saint Louis Code for America Brigade, OpenSTL. This project is open source and is free to be reused and distributed.

The files can be found on the OpenSTL Backyard Chickens GitHub Page
